
# ClinicBooking API (MVP) — .NET 8 + EF Core + PostgreSQL

API de prise de rendez-vous (clinique) en ASP.NET Core (.NET 8) avec EF Core + PostgreSQL, Swagger, validation (FluentValidation), migrations automatiques et seed de données.

---

## Stack
- .NET 8 (ASP.NET Core Web API)
- EF Core 8 + Npgsql
- PostgreSQL (Docker)
- Swagger / OpenAPI
- FluentValidation

---

## Run en 30 secondes (Docker)
Pré-requis : Docker Desktop

### Démarrer
```bash
docker compose up --build
